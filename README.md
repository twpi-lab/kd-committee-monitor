# 위원회/국민참여단 모집 공고 알림봇

공개된 공고 게시판을 주기적으로 확인해서 위원회, 국민참여단, 정책모니터단, 시민감사관 등 실제 참여 가능한 모집 공고를 텔레그램으로 알려주는 봇입니다.

## 현재 모니터링 범위

### 지역/광역

1. 양주시 고시공고 — 거주 근거
2. 의정부시 고시공고 — 직장 사업자 주소 근거
3. 경기도 고시공고 — 경기도 거주 근거
4. 남양주시 고시공고 — 사무실 주소 근거
5. 성동구 고시공고 — 서울 사무실/직장 소재 근거
6. 경기도의회 공고 — 경기도 거주/광역 참여 가능성

### 전국 단위, B안 확장

7. 청년DB — 정부위원회 청년위원 후보/인재풀 성격
8. 국가교육위원회 — 국민참여위원회 등
9. 국민권익위원회 — 청렴시민감사관, 국민참여단, 정책모니터단 등
10. 국민통합위원회 — 국민참여단, 자문단, 모니터단 등
11. 국민참여예산 — 국민참여예산 관련 위원/지원단 등

서울시는 사용자 결정에 따라 제외했습니다.

## 매칭 규칙

기본 규칙은 다음과 같습니다.

`위원회/참여단 유형 키워드` AND `모집 신호 키워드` - `제외 키워드`

예시 매칭:

- 국민참여위원회 위원 공개모집
- 정책모니터단 참여자 모집
- 청렴시민감사관 공개모집
- 정부위원회 청년인재 추천 및 인재풀 등록

예시 제외:

- 선정 결과
- 회의 개최
- 회의록
- 입찰/용역
- 채용
- 이벤트
- 공모전 결과

## GitHub Actions 설정

Repository Settings → Secrets and variables → Actions에 아래 Secrets를 등록하세요.

- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`

그 뒤 Actions 탭에서 `위원회 모집공고 모니터링` 워크플로우를 수동 실행하면 됩니다.

## 로컬 실행

```bash
pip install -r requirements.txt
python -m playwright install chromium
python main.py --once
```

## 첫 실행 후 확인할 것

실행 로그에서 각 사이트별로 다음 형식이 나옵니다.

```text
📋 사이트명: N건 확인
```

0건이 계속 나오는 사이트는 게시판 구조가 달라졌거나 JavaScript 렌더링이 필요한 경우일 수 있습니다. 그때 해당 사이트만 `parser: "playwright"`로 바꾸거나 전용 파서를 추가하면 됩니다.
